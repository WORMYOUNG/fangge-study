# 方歌学习测试工具

北京中医药大学 25级王琦班 · 杨智行 开发

## 在线访问
https://yangqixi.cn/ft/

## 功能
- 按章节分类学习和测试方歌
- 本地记录掌握进度，方便查漏补缺
- 静态页面，无需后端，打开即用

## 数据来源
- **方歌内容**：北京中医药大学 方剂学课程（授课老师整理）
- **程序开发 / JSON 数据转换 / 前端逻辑**：杨智行

## 使用说明
- 推荐在电脑端浏览器打开
- 测试进度保存在浏览器本地存储，关闭页面后清零
- 方歌数据经多次核对，如有问题欢迎反馈
- 本工具不收集存储任何用户信息
